GlobaLeaks Bug Hunting

Kill bugs, get swag and 1337 credz!

Make GlobaLeaks more secure!

We invite hackers and infosecurity communities to play with globaleaks source code as a way to spot security bugs. We will be happy to rewards all security bug's reporters by sending an Hermes Center branded USB key with Tails pre-loaded, and a mention on our website!

Security bugs we are looking for

We're looking for any bug that maliciously impacts the security of our users, within the boundaries of our threat model. Implementation bugs, XSS, anonymity or privacy weaknesses, code execution and so on.

How to report bugs?

You can file a bug report on our GitHub repository. If you'd rather be private, mail us at info at globaleaks org. with PGP and we will take care of discussing and fixing the issues!
