GlobaLeaks Bug Hunting

Make GlobaLeaks more secure!

To make GlobaLeaks safer, we invite hackers and infosecurity experts to play with globaleaks source code helping the communty to spot security bugs. We will be happy to rewards all security bug's reporters by sending an Hermes Center branded USB key with Tails pre-loaded and a mention on our website!

Security bugs we are looking for

We're looking for any bug maliciously impacts the security of our users, within the confines of our threat model. implementation bugs, XSS, anonymity or privacy weakness, ,code execution and so on.

How to report bugs?

You can file a bug report on our GitHub repository. If you'd rather be private mail us at info at globaleaks org. with PGP and we will take care of discussing and fixing the issues!
