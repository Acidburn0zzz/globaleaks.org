GlobaLeaks Bug Hunt

Help make GlobaLeaks safer, get rewards and eternal greatness!

May 2013 GlobaLekas is launching a bug hunt and we need your help! Look through the source code and squish security bugs. You'll be helping make free, open source software more secure and you'll get a USB key, Hermes Center branded with Tails pre-loaded and a mention on our Wall of Unquestionable Greatness!

What we're looking for

We're looking for any bug maliciously impacts the security of our users, within the confines of our threat model. XSS bugs, crypto implementation bugs, arbitrary code execution and so on.

How do I report bugs?

We strongly believe in full disclosure. Please file a bug report on our GitHub repository. If you'd rather be private (there's no need!) email info @ globaleaks org.
